pour executer les codes Meteor sous windows:
-telecharger Meteor pour windows: https://www.meteor.com/install
-ouvrir un cmd
deplacer dans le fichier tuto : cd /--------/tuto
tapez: metor
- copier le localhost
- coller-le dans le navigateur
- taper: entrer

pour remplir les bases de données: 
ouvrir un autre cmd et taper:
- meteor mongo
- db spots.insert ({'text' ou bien 'name':'-------un texte----'});
- db spots.insert ({'text' ou bien 'name':'-------un autre texte----'});

	- db spots.find ({'votre texte'});
