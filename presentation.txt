L’objectif de ce projet est de développer un site web qui affiche en temps réel les données de navigation d’un kite surfeur. Le second axe du projet consistera en un développement de plusieurs fonctions sociales (partage, commentaires, …) autour de ces données. Enfin, il faudra aussi développer une solution intelligente pour adapter le site web aux smartphones. 
Les utilisateurs disposeront de privilèges tels que consulter les classements des meilleurs prestations de surf, ajouter des données multimédias etc...
Réaliser un tel projet nécessitera des connaissances approfondies sur les langages web tels qu’HTML5, CSS, JavaScript ainsi que sur la bibliothèque Meteor.js qui est un framework open-source de développement web en JavaScript basé sur Node.js. Il faudra aussi des compétences en Gestion de Base de données et une connaissance assez poussée en Réseau.

Dans le dossier 'tuto' vous trouverez la partie realisee jusqu'a present.
